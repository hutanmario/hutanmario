# Hi, I'm Mario

I build full-stack applications with a strong interest in AI-assisted products, recommendation systems, and mobile experiences.

My current flagship project is **TripCraft**, an AI-assisted travel planning app that combines:

- React Native and Expo for the mobile experience
- FastAPI, SQLAlchemy, PostgreSQL, and Alembic on the backend
- CLIP-based image understanding for photo onboarding
- Personalized recommendation and itinerary generation
- Evaluation scripts for recommendation quality and image-recognition metrics

## Featured Project

### TripCraft

An intelligent travel planner that builds a traveler profile from an adaptive quiz or selected travel photos, recommends destinations, and generates personalized multi-day itineraries.

Tech: `React Native` `Expo` `FastAPI` `PostgreSQL` `SQLAlchemy` `PyTorch` `Transformers` `CLIP`
